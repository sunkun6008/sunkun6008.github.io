生信环境建立————bioconda的安装与使用
学习 linux bioconda
学习 linux 2018-11-24

前言

一般生信分析都需要必要的生信分析环境，环境如何建立呢？如果你不想折腾，想要直接就可以使用，建议安装bioLinux系统，里面有很多的生信必须的分析软件，解决了依赖，变量以及环境的问题，当然如果你想在学习生信的基础上学习一些Linux的一些知识和网页构架的一些知识，建议使用ubuntu 系统，自己安装建立生信环境，毕竟BioLinux系统也是建立在ubuntu 14.0的基础之上的，建立生信分析环境的变量有很多，这里简单介绍bioconda。

正文

bioconda安装

wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh -b -p $HOME/Miniconda3

修改环境

sudo gedit /home/sk/.bashrc
添加export PATH=$HOME/Miniconda3/bin:'$PATH'
或者使用 echo  "export PATH=$HOME/Miniconda3/bin:"'$PATH' >> ~/.bashrc source ~/.bashrc

添加清华源镜像

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/bioconda
conda config --set show_channel_urls yes

创建生信分析环境

conda create -y --name bioinfo python=3

激活环境

source activate bioinfo
    conda info --envs #查询环境
    source deactivate #关闭

批量安装生信分析用到的软件

curl http://data.biostarhandbook.com/install/conda.txt | xargs conda install -y

批量更新

curl http://data.biostarhandbook.com/install/conda.txt | xargs conda upgrade -y

软件安装

conda install R
conda install Rstudio
