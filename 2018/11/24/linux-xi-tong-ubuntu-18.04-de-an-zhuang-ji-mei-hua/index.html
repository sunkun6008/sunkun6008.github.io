Linux系统————ubuntu 18.04的安装及美化
Linux ubuntu 学习
linux 2018-11-24

前言

一般的生信操作系统都会建立在Linux的系统之上，当然目前建立在Linux所提供的核心基础之上进行开发的系统有很多，目前来说，总共分为两大派系，Deb 以及 Red，一般服务器使用的都是Red

不过按照我个人的建议，如果是个人计算机，建议安装ubuntu，如果是个人服务器或者部分公司建议安装CentOS,毕竟是免费的，如果有钱，又渴望稳定还是安装RedHat吧

很明显小编没有money，所以喽,只能使用个人计算机版的东西，我会以ubuntu为基础进行建立。

正文

网易云音乐修复

sudo gedit /etc/sudoers

修改/etc/sudoers文件，加一行：’YOURNAME ALL = NOPASSWD: /usr/bin/netease-cloud-music’
YOURNAME为你登录的用户名

sudo gedit /usr/share/applications/netease-cloud-music.desktop

修改Exec=netease-cloud-music %U 为 Exec=sudo netease-cloud-music %U
这样点击网易云音乐图标就是以管理员权限启动的了，且不用输入密码了。
于此同时设置root/admin等没有密码登录

18.04 键盘背光打开

建立led.sh文件，代码如下：

#!/bin/bash - 
#===============================================================================
#
#          FILE: ledctrl
# 
#         USAGE: ./ledctrl

#   DESCRIPTION: 
# 
#       OPTIONS: ---

​
status=/tmp/keyboard_led_status
if [[ ${1} == "on" ]] || [[ ! -e ${status} ]]; then
    xset led named 'Scroll Lock' && echo "on" > ${status}
else
    xset -led named 'Scroll Lock' && rm -f ${status}
fi

于 /bin中新建一个led文件，将led.sh放入文件中，于键盘中添加快捷键，命令为 bash /bin/led/led.sh,设置快捷键即可。

为知笔记的安装

必要软件安装
