R语言基础--R的安装使用及基础语法
基础 学习 R语言
R语言 2018-12-13

前言

现在社会，只要是专业的统计学领域，几乎都听过且使用过R，那么问题来了，什么是R呢？通过本篇我会带大家基本了解一下什么是R，应该如何安装，又存在哪些基本的语法。

正文

1、什么是R？

R即是软件，也是语言，在GNU协议下免费发行，是1995年由新西兰奥克兰大学统计系的Ross Ihaka 和Robert Gentleman 基于S语言基础上共同开发的一种统计软件。现在由R开发核心小组负责维护与更新，并将全球优秀的统计应用程序包免费提供给大家使用、共享。总的来说，R是一套由数据操作、计算和图形展示功能整合而成的软件系统。
R具有以下几个显著的优点：

  • 免费且开源
  • 运算快且功能强大
  • 操作系统依赖性低
  • 帮助功能完善
  • 作图功能强大
  • 统计分析能力突出
  • 兼容性强
  • 强大的拓展与开发能力

和大多数编程语言一样，R语言编程的核心是编写“函数”。所谓函数就是一组指令的集合，用来读取输入、执行计算、返回结果。所以学习R就是不断的学习一个有一个函数的过程，不过万丈高楼终须从平地而起，对于R来说，平地就是基本的语法结构，，以及规定的语言结构。

2、R安装及R语言操作界面——Rstudio

如果你的操作系统是Windowns系统，那么就很简单了，直接进入官网下载运行即可。下面是官网提供的下载地址： R下载页面 进入到本页面选择不同的镜像下载即可。
如果你的操作系统是基于unix的或者Linux的那么你需要打开终端，输入

apt-get install r

当然你也可以建立生信分析环境，这样就不用为如何安装R而烦恼了。建议安装conda,是一种很全的生信软件管理器，可以自动创建环境。
言归正传，以Windowns 界面为例接着讲解如何安装R，R下载完毕后和常规exe文件类似，直接运行即可，没有难度。
此处再推荐一个优秀的R语言操作界面——Rstudio,下载地址 ，进入页面选择适合你操作系统的版本安装即可。在此再多说一句，如果你是使用Linuc操作系统，并且已经安装了conda，你只需要在conda的终端环境中输入：

conda install rstudio

很快就可以直接使用了，并且很多基础包都已经下载完毕了。通过Rstudio运行R具有以下几个优点：

可兼容多个版本的R软件
代码字体高亮，代码完整性智能识别、自动缩进
可直接执行R程序代码
可直接浏览工作表和数据
可随意缩放绘制的图形，并且有多种输出格式
整合R帮助和R使用文档
可查看R命令的运行记录

3、R语言的基础语法

下面重点来了！！醒醒了各位小伙伴。下面我会给大家简单介绍R语言的基础语法。总的来说R语言的基础语法主要包括以下五个部分：

  • 对象与变量
  • 运算符
  • 表达式
  • 特殊值
  • 控制结构

3.1 对象与变量

R代码就是操作于对象，简单理解，对象就是R代码里的个体，可以包含向量、列表和函数等。在R中，将变量的名字称为符号。当一个对象赋值给某个变量时，就是将该对象赋给当前环境里的一个符号，在函数中这个符号也被称为形参，即形式参数，这是相对于实际参数，实参而来的，也可以说变量就是一个可变量，是一个形式参数，也仅仅是个占位符。例如：

>x<-1

表示把数值1（对象）赋给符号X（变量），‘<-’代表赋值符号。变量一般是给定的某个符号，在函数中依据位置的不同，又将变量分为局部变量和全局变量。局部变量是指只在函数体内部可见的变量，全局变量则是指在函数之外创建的变量，当然全局变量在函数内部也可以访问。一般来说修改某个或某些变量名，采用编辑器或使用函数的方式。

3.2 运算符

运算符operator是包含一个或两个参数的无括号的函数。常用的运算符包括算术运算（加、减、乘、除）、取模运算、指数运算、整除运算以及比较运算等。不同的运算符其优先级别各不相同。

3.3 表达式

表达式expression是由数字、算符、数字分组符号（括号）、自由变量和约束变量等以能求得数值的有意义排列方法所得的组合。在R中可以用分号、括号和花括号来组织表达式。

3.4 特殊值

在R中，特殊值主要有四类，首先来介绍一下缺失值，缺失值以符号NA（Not Available，不可用）表示，函数is.na()可检测缺失值是否存在，在某些运算中可以设置na.rm=T,来去除缺失值，同时还有oa.omit()可以移除所有含有NA数据的行。第二种特殊值是非数值（NaN,Not a number),与NA不同，表示一个没有意义的结果。第三种特殊值是Inf和-Inf分别表示正无穷和负无穷，当R计算得到一个很大的数值时，也会返回Inf或-Inf。第四种特殊值为NULL，表示空对象，可被用于赋给一个对象，常用于函数的参数中，表示该参数未被赋值，NULL也可以用于删除数据框中的变量。

3.5 控制结构

控制语句是用来实现对程序流程的选择、循环、转向和返回等进行控制。在R中可分为条件语句和循环语句。
条件语句主要包括：

if/else 语句
ifelse 语句
switch 语句

循环语句主要有：

for 循环
while 循环
repeat 循环

if/else 语句

if/else语句用来进行条件控制，以执行不同的语句。使用格式如下：

if(condition){exprl} else {expr2}

如果condition条件为真，则执行expr1，否则执行expr2。此外，操作符&&和||经常被用于条件部分，&和|是按照逐个元素的方式进行计算，而&&和||则对向量的第一元素进行计算，只有在必需的时候才对第二个参数进行运算。

ifelse 语句

ifelse语句是if/else结构的向量版本，其使用格式为：

 ifelse(condition,a,b)

ifelse语句的执行规则是：如果condition为真，则执行a语句；反之对应的是b语句。

switch 语句

switch语句是多分支语句，其使用格式为：

switch(expr,list)

其中expr是表达式，list是列表。

for 循环

最基本的循环为for循环，其使用格式如下：

for(n in x){expr}

其中，n为循环变量，x通常是一个序列。每次循环时，n从x中按顺序取值，并代入到后面的expr语句中进行运算。

while 循环

当不确定循环次数时，则用while循环语句。其格式如下：

while(condation){expr}

在condation条件为真时，执行大括号内的expr语句。

repeat 循环

当不能确定循环次数时，还可用repeat循环语句，其格式如下：

repeat{(condation1)expr;(condation2)break}

在condation1条件为真时，执行expr语句，直至condation2条件为真时，执行break语句终止repeat循环。
虽然循环语句看起来比较便利，但涉及计算量比较大时，一般不适用循环，而改用向量或矩阵运算，后者的运算速度要远快于循环。
